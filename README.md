# Agent Control Center

Nuxt 4 dashboard project configured for Claude Code-first development.

## Claude layers
- `CLAUDE.md` for core project instructions
- `.claude/rules/` for enforceable development behavior
- `.claude/skills/` for repeatable workflows
- `.claude/hooks/` for lifecycle automation
- `.mcp.json` for project MCP servers
- `docs/` for human-readable context
