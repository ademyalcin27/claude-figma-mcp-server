# Architecture

## App side
The UI lives under `app/`.

## Server side
Mock and real API routes live under `server/api/`.

## Shared side
Common types, constants, and schemas live under `shared/`.

## Dashboard shape
The first version includes:
- Overview
- MCP Servers
- Skills
- Workflows
- Logs

## Evolution path
1. mock routes
2. typed UI
3. reusable components
4. validation and schemas
5. real integrations
